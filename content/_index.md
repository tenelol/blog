Go / Nix / CLI / Hugo を使いながら、個人開発と運用改善を継続しています。インターン向けに、作ったものだけでなく、どう整理し、どう改善したかまで追えるようにまとめています。

このサイトでは `Works` を制作物、`Writing` を技術メモと記事導線、`About` をプロフィールと関心領域に分けています。

## What I Want You To See

- 再現性を意識した開発と運用
- 小さなツールや仕組みを自分で切り出して作る姿勢
- 実装だけでなく、設計意図や判断も文章で残す習慣

## Main Areas

- Go と Nix を使った小さな Web / CLI 実装
- NixOS / nix-darwin を含む個人環境の再現性ある管理
- 個人サイトや技術記事を通した継続的な発信
