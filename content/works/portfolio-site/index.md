このサイト自体を、`Works` セクションのひとつの制作物として掲載しています。

## Goals

- 制作実績とブログをひとつのサイトで運用する
- プロフィールから各コンテンツへの導線を明確にする
- 記事や実績を追加しやすい構成に保つ

## Visual Direction

- 手元の夜桜写真をキービジュアルとして使い、静かな雰囲気を最初に伝える
- Blowfish の `background` レイアウトで、背景画像に下方向のブラーをかける
- GitHub の制作物と Zenn の記事を同じ導線で回遊できるようにする

## Stack

- Hugo
- Blowfish
- GitHub Actions
- Tailwind CSS
