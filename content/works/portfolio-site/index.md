このサイト自体を、`Works` セクションのひとつの制作物として掲載しています。

## Goals

- 制作実績とブログをひとつのサイトで運用する
- プロフィールから各コンテンツへの導線を明確にする
- 記事や実績を追加しやすい構成に保つ

## Visual Direction

- 手元の夜桜写真をキービジュアルとして使い、静かな雰囲気を最初に伝える
- `hero` レイアウトでトップの印象を強めつつ、直下に `Works` と `Posts` を並べる
- 情報量を増やしても崩れにくいカードベースの一覧を採用する

## Stack

- Hugo
- Blowfish
- GitHub Actions
